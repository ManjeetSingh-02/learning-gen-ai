# Prompting

- Zero Shot Prompting: Direct Instructions
- Few Shot Prompting: Direct Instructions with some examples
- Chain of Thought Prompting: Direct Instructions with some examples and reasoning steps
- Role Playing Prompting: Direct Instructions with some examples and reasoning steps, where the model is asked to take on a specific role

## Types of Prompting
- Alpaca (Alpaca Models): Markdown format
- ChatML (OpenAI ChatGPT): JSON format
- INST (LLaMA2 Chat): [INST] This is INST format [/INST]
- FLAN-T5 (Google Gemini): "Question: What is AI?\nAnswer:"
